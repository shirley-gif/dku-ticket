DKU Library Ticket Chat

Hello, if you want to raise a ticket about DKU Library Systems, you are in the right place.
Steps to creat a ticket:
- 1. Input your DKU email (the one ended with dukekunshan.edu.cn)
- 2. Click 'Start' button to begin the chat
- 3. Follow the chat and input details about the question
- 4. Input 'restart' or 'cancel' if you want to start over from step 1.
- 5. No ticket will be created until you type confirm.
Confirmation email and follow up emails will be sent to you after the ticket is confirmed.

Web App

Input email and click start

Commands: restart (start over) / cancel (stop)
Validation rules:
• Title: 5–120 characters
• Description: at least 15 characters
• Urgency / Impact: Low / Medium / High
• System: Alma / Summon / AtoM / Archivematica / RFID / Other
Tip: If Ping never returns, you may be opening an old deployment. Please redeploy and open the /exec URL.
When you see the confirmation summary, type confirm to submit the ticket request.
